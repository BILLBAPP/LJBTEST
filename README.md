# LJB Security Training — Website Redesign Prototype

A two-page redesign concept for [ljbsecuritytraining.com](https://www.ljbsecuritytraining.com), built by Media Room CT.

## Pages
- **index.html** — Homepage: hero, live class roster, 3-step path, course catalog, packages, About Lou, reviews
- **training.html** — Programs & Pricing: full class pricing table, bundles (current + 3 proposed new tracks), advanced career path map, and recommended training tracks for churches, HR, business owners, schools, and security leadership

## Notes for review
- Single-class prices shown at the $150 standard rate implied by current package math — confirm and adjust per class before launch
- "Ask Lou" bundles (Armed Officer Track, Hospital & Campus Ready, Leadership Track) are **proposed new offerings**, not current products
- Class dates in the roster are placeholders — final build wires these to a real calendar/booking system
- Photo placeholder in About section is reserved for a shoot of Lou teaching a live class

## Deploy on GitHub Pages
1. Create a new repository (e.g. `ljb-redesign`)
2. Upload all files/folders in this directory to the repo root
3. Repo **Settings → Pages → Source: Deploy from a branch → main / (root) → Save**
4. Site goes live at `https://<your-username>.github.io/ljb-redesign/` within a minute or two

No build step, no dependencies — plain HTML/CSS/JS.
